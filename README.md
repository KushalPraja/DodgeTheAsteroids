# dodge the asteroid

this is a simple game.\
just dodge the asteroids.\
beat your high score.



## how to setup

1. install uv (if you don't have it)
    ```bash
    uv --version

2. clone the repo 

3. install dependencies
    ```bash
    uv sync
    ```

4. run the game
    ```bash
    uv run python main.py
    ```

5. have fun!
